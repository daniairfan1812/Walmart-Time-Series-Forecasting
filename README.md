# Walmart-Time-Series-Forecasting
Time series data preparation and analysis project using Python

## Overview

This project focuses on preparing and analyzing time series data for forecasting using Walmart sales data.

The dataset contains weekly sales information of Walmart stores along with factors such as temperature, fuel price, CPI and unemployment rate.

## Dataset

- Records: 6435
- Frequency: Weekly
- Main Variable: Weekly Sales

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Project Workflow

1. Importing libraries
2. Loading dataset
3. Converting date into datetime format
4. Setting date as index
5. Resampling weekly sales into monthly sales
6. Calculating moving averages
7. Rolling mean and standard deviation analysis
8. Visualizing sales trends
